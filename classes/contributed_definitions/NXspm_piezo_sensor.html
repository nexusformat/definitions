3.3.3.87. NXspm_piezo_sensor¶

Status:

base class (contribution), extends NXsensor

Description:

This piezo sensor group refers to the height (or Z) piezo sensor.

Symbols:

No symbol table

Groups cited:

NXspm_piezo_config, NXspm_piezoelectric_material, NXspm_positioner
