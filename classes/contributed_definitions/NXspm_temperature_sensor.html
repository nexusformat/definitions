3.3.3.93. NXspm_temperature_sensor¶

Status:

base class (contribution), extends NXsensor

Description:

The :ref:`NXspm_temperature_sensor` class describes a temperature sensor ...

The NXspm_temperature_sensor class describes a temperature sensor used in scanning probe microscopy (SPM) experiments. It includes fields for the temperature measurement, calibration, and associated data.

Symbols:

No symbol table

Groups cited:

NXcalibration, NXdata, NXparameters

Structure:

temperature: (optional) NX_NUMBER {units=NX_TEMPERATURE}

The obtained average temperature of the sensor.

target_temperature: (optional) NX_NUMBER {units=NX_TEMPERATURE}

The target temperature of the sensor, which is the desired temperature ...

The target temperature of the sensor, which is the desired temperature to be maintained during the experiment.

temp_offset_value: (optional) NX_NUMBER {units=NX_TEMPERATURE}

The offset temperature of the sensor, which will be added to obtained correct ...

The offset temperature of the sensor, which will be added to obtained correct the measured temperature.

TEMPERATUREchannel: (optional) NX_CHAR

The name of the channel that handles the temperature measurement.

calibration: (optional) NXcalibration

Calibration of the temperature measurement.

calibration_parameters: (optional) NXparameters ⤆

coefficient: (optional) NX_NUMBER {units=NX_ANY}

The coefficient of the calibration.

DATA: (optional) NXdata ⤆

Data related to the temperature measurement, such as time series data ...

Data related to the temperature measurement, such as time series data or voltage-temperature data.

DATA: (optional) NX_NUMBER {units=NX_TEMPERATURE} ⤆

Temperature data collected during the scan.

AXISNAME: (optional) NX_NUMBER {units=NX_ANY}

independent axis data like time or position or bias voltage.
