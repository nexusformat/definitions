==============
NXspectrum_set
==============

..
    Contributors List

**Status**:

  base class, extends :ref:`NXobject`

**Description**:

  Container for reporting a set of spectra.

**Symbols**:


  **n_y**: Number of pixel in the slow direction.

  **n_x**: Number of pixel in the fast direction.

  **n_energy**: Number of energy bins.
